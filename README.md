News Article Text Analysis Tool

A Python program that performs various text analysis tasks on news articles.

Files
->python_assessment.py - Main Python script with analysis functions
->news_article.txt - Text file containing the news article to analyze

Requirements
--Python 3.x and above works best.

No additional packages needed (uses only built-in modules)


How to Run
Make sure both files are in the same directory:

Open terminal/command prompt in that directory

Run the script:

python python_assessment.py
